如何用这个？

第一，如果你说英语，下载 clean(English).py；
如果你说中文，下载 clean(Chinese).py。
把它们放在根目录。

第二，搜索 http://github/termux 并下载 Termux。
或者从我的库下载termux

第三，在 Termux 上安装 Python：
运行 pkg update && pkg upgrade
运行 pkg install python

第四，连接存储权限：
运行 termux-setup-storage
然后输入"y"

运行 python 你下载的文件.py

最后，开始使用！
